
# SpaceX Internal Search Tool

A (very simple) CLI to search SpaceX's GraphQL API for some interesting information


## Installation

Clone this repository

```bash
  git clone https://github.com/christiaankruger/KB4_BE
```

Install Gems

```bash
bundle install
```

## Usage

```bash
  bin/spacexsearch -s SEARCH_TERM
```

e.g.

```bash
  bin/spacexsearch -s "NASA"
  bin/spacexsearch -s "Dragon"
  bin/spacexsearch -s "Space"
```
